SMS sending Widget for your site

== Description ==

Very easy to use plugin for Wordpress. Add a widget to your website for sending SMS messages. Sending to all phones in the world. 

== Installation ==

1. Upload the plugin sms files to the `/wp-content/plugins/` directory.
2. Activate the SureSMS plugin through the 'Plugins' menu in WordPress.
3. Fill out the form on SureSMS.com to get username and password.
4. Add the username and password on setting page.
5. Add Footer Text to be send in every SMS.
6. Drag and Drop SMS widget to you sidebar from Apperance > Widgets OR can use [SMS] shortcode to keep in post and page.
7. You are ready to allow you site vistors to send Free SMS.

== Frequently Asked Questions ==

= Can i add more SMS credits to same login key ? =

Yes, you can.

= Do i need to use all sms with in time limit ? =

No, you can use it as your need. No time bound for it.
